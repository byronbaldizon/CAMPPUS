La energía interna (U)

Los sistemas están compuestos por partículas, es decir, átomos y moléculas. Estas partículas tienen masa y sabemos que se mueven. Esto demuestra que las partículas tienen energía cinética. Esta energía está almacenada dentro del sistema.

Le energía interna (U) es la energía almacenada dentro de un sistema almacenada en sus partículas.

El calor (Q)

El calor es un flujo de energía. A diferencia de la energía interna, el calor es energía en movimiento, energía en tránsito que fluye de un punto del universo a otro debido a una diferencia de temperatura.

El trabajo (L)

A diferencia del calor, el cual se origina debido a una diferencia de temperatura, el trabajo es un flujo de energía provocado por cualquier causa que no sea una diferencia de temperatura. Este sería el caso cuando aplicamos una fuerza sobre una masa. En este caso se produce un flujo de energía hacia el cuerpo ocasionado por una fuerza. O cuando una carga eléctrica se mueve bajo la acción de un campo eléctrico. En este caso la causa del movimiento de la carga es el campo eléctrico. Se dice entonces que hay un trabajo eléctrico.

Los sistemas termodinámicos

La Termodinámica considera que un sistema termodinámico es la parte del Universo que se está estudiando.El entorno de un sistema es todo aquello que queda por fuera del sistema pero que está contenido en el Universo.

Los sistemas termodinámicos se clasifican en:

  • Abiertos: Son sistemas que pueden intercambiar energía y masa.
  • Cerrados: Son sistemas que pueden intercambiar energía y pero no pueden intercambiar masa.
  • Aislados: Son sistemas que no pueden intercambiar ni calor ni trabajo.

El estado termodinámico

El estado termodinámico de un sistema es el conjunto de los valores de presión, volumen y temperatura. Por esto, al estado termodinámico de un sistema se le conoce como estado (P,V,T).

Para estudiar los estados termodinámicos de un sistema y sus evoluciones o procesos se acostumbra a representar al sistema como un punto en el diagrama presión-volumen o diagrama PV.

Los procesos termodinámicos

Un sistema termodinámico puede cambiar de estado termodinámico. Este cambio se conoce como proceso termodinámico.Algunos procesos termodinámicos característicos son:

  1. Proceso isotérmico: Es un proceso en el que el sistema mantiene una temperatura constante.
  2. Proceso isobárico: Es un proceso en el que el sistema mantiene una presión constante.
  3. Proceso isocórico: Es un proceso en el que el sistema mantiene un volumen constante.
  4. Proceso adiabático: Es un proceso en el que el sistema no intercambia calor.

El Primer Principio de la Termodinámica

Sabemos que la energía puede fluir, quedarse almacenada y transformarse de un tipo de energía a otro. Durante estos flujos y transformaciones cantidad de energía no cambia. Es decir, la energía no se crea ni se transforma. Solamente se transforma.

Para sistemas cerrados,la expresión matemática del primer principio de la termodinámica es:

ΔU=Q-L

Convención de Signos

Para poder utilizar la expresión matemática de la primera ley debemos observar la siguiente convención de signos con respecto al calor y al trabajo:

  • Calor (Q): Si el calor entra al sistema su signo será positivo. Si el calor sale de un sistema, su signo será negativo.
  • Trabajo (L): Si el trabajo entra al sistema su signo será negativo. Si el trabajo sale de un sistema, su signo será positivo.
  • Variación de Energía Interna (ΔU): Si la energía interna interna de un sistema aumenta entonces ΔU será positivo. Si la energía interna de un sistema disminuye entonces ΔU será negativo.

El trabajo PV

Existe un tipo especial de trabajo conocido como trabajo PV (LPV) o trabajo de expansión-compresión. Este trabajo es el que se manifiesta durante los cambios de volumen de un sistema.

Cálculo del trabajo PV

Método integral

El trabajo PV puede calcularse por medio de la integral:

LPV=∫PdV

Método gráfico

El área bajo la curva del proceso estudiado es el trabajo PV.

Fórmulas

Si el sistema que estamos estudiando es un gas ideal, existen fórmulas que nos permiten calcular el valor del trabajo PV de cada proceso. Ver Termodinámica de gases ideales.

Funciones de estado

El valor de la energía interna de un sistema se puede calcular si se conoce el estado PVT de dicho sistema. En otras palabras la energía interna es una función de estado porque su valor sólo depende del estado del sistema y no de su historia previa. Entonces, cada punto del diagrama PV representa un valor determinado de energía interna del sistema, pues cada punto representa un estado PVT de dicho sistema.

Por otro, el trabajo de un proceso termodinámico no es una función de estado porque su valor depende del proceso que se esté realizando.

Finalmente podemos decir que, al igual que el trabajo,el calor de un proceso termodinámico no es una función de estado porque su valor también depende del proceso que se esté realizando.

Los procesos cíclicos

Un proceso cíclico es aquél en el cual el estado final es igual al estado inicial. En un proceso cíclico el sistema regresa al estado inicial luego de haber transitado por diversos estados intermedios.

Cuando un sistema evoluciona cíclicamente, el sistema parte de un estado con determinado valor de energía interna y regresa al mismo estado. O sea que regresa al mismo valor de energía interna que tenía al principio. Entonces, la variación de energía interna del sistema de un sistema que describe un proceso cíclico valdrá cero:

ΔU = Ufinal - Uinicial

Pero como el sistema regresa al estado inicial entonces se cumple que:

Ufinal = Uinicial

Por lo tanto:

ΔUciclo = Uinicial - Uinicial = 0
